# Content Types — Format Definitions

## Primary Formats

| # | Type | Purpose | Best On |
|---|------|---------|---------|
| 1 | Pain Point Post | Make reader feel unquantified problem | X, LinkedIn, IG |
| 2 | Case Study | Social proof with specific numbers | LinkedIn, X, YouTube |
| 3 | How-To / Tutorial | Teach, establish expertise | YouTube, IG carousel, X thread |
| 4 | Competitor Comparison | Capture decision-stage buyers | YouTube, LinkedIn, X |
| 5 | Hot Take / Contrarian | Spark debate, drive engagement | X, Threads, LinkedIn |
| 6 | Data / Research | Original insights that get shared | LinkedIn, Newsletter, X |
| 7 | Behind-the-Scenes | Humanize the brand | TikTok, IG Stories, Newsletter |
| 8 | Listicle | Scannable value, high save rate | IG carousel, X thread |
| 9 | Community Discussion | Start conversations | Facebook, LinkedIn, Threads |
| 10 | Product Demo | Show don't tell | YouTube, TikTok, X video |
